사용법

세 단계로 끝납니다.
설치는 필요 없습니다.

01

PPTX 파일을 올립니다

파일을 드롭존에 끌어다 놓거나, 드롭존을 클릭해서 파일을 선택합니다. .pptx 형식만 지원합니다. 파일 크기 제한은 없으며, 브라우저 메모리가 허용하는 한 처리됩니다.

02

탭을 선택해서 에셋을 확인합니다

파일이 로드되면 폰트·이미지·색상·벡터 탭이 나타납니다. 각 탭의 숫자는 추출된 항목 수입니다. 탭을 눌러 원하는 에셋을 확인합니다.

03

필요한 에셋을 가져갑니다

이미지는 개별 내려받기 또는 ZIP 일괄 다운로드가 가능합니다. 색상은 클릭하면 HEX 코드가 클립보드에 복사됩니다. 다른 파일을 보려면 상단의 "다른 파일 열기"를 누릅니다.

탭별 기능 설명

폰트

사용된 폰트 목록

슬라이드 XML에서 폰트 이름을 모두 추출해 나열합니다. 파일 안에 직접 포함된 폰트는 "포함됨" 배지로 표시됩니다. 포함된 폰트는 PPT를 열지 않아도 이름을 확인할 수 있습니다.

이미지

래스터 이미지 추출

PNG·JPG·GIF·BMP·WEBP 파일을 미리보기와 함께 보여 줍니다. 파일명·해상도·용량이 표시되며, 이미지별 내려받기와 전체 ZIP 다운로드를 지원합니다.

색상

테마 색 + 사용된 색

테마의 공식 팔레트(dk1, lt1, accent1 … 등 12개 슬롯)와 슬라이드에서 실제로 쓰인 색상을 사용 빈도 순으로 나눠 보여 줍니다. 색상 칩을 클릭하면 HEX 코드가 복사됩니다.

벡터

SVG / EMF / WMF 파일

SVG는 브라우저에서 바로 미리보기가 가능합니다. EMF·WMF는 브라우저 렌더가 되지 않아 미리보기 없이 파일 다운로드만 제공됩니다.

더 효율적으로 사용하는 팁

01

색상 코드는 하나씩 클릭해서 복사

색상 칩을 클릭할 때마다 HEX 코드가 클립보드에 덮어씌워집니다. 여러 색상을 정리하려면 하나씩 클릭해서 메모장이나 피그마에 붙여넣는 방식이 가장 빠릅니다. 테마 팔레트 섹션에서 주요 브랜드 색상을 먼저 확인하세요.

02

이미지는 ZIP으로 한 번에 내려받기

ZIP 일괄 다운로드를 사용하면 슬라이드에 삽입된 이미지가 원본 해상도 그대로 저장됩니다. PPT에서 이미지를 개별로 "다른 이름으로 저장"하는 것보다 원본 품질을 유지하는 데 유리합니다.

03

"포함됨" 배지가 붙은 폰트를 주목

폰트 탭에서 "포함됨" 배지가 붙은 폰트는 PPT 파일 안에 직접 임베드된 폰트입니다. 해당 폰트가 내 컴퓨터에 설치되어 있지 않아도 파일 제작자가 어떤 폰트를 사용했는지 이름으로 확인할 수 있습니다.

04

다운로드가 막힌다면 팝업 차단 해제

브라우저 팝업 차단 설정이 파일 다운로드를 막을 수 있습니다. 주소창 오른쪽의 팝업 차단 아이콘을 클릭해 허용으로 변경하면 이미지 개별 다운로드와 ZIP 일괄 다운로드가 정상 작동합니다.

참고 사항
  • PPT 자체 도형(텍스트 박스, 도형 등)은 추출되지 않습니다. ppt/media/ 폴더에 있는 파일만 대상입니다.
  • 오디오·비디오 파일은 지원하지 않습니다.
  • 파일이 크거나 슬라이드 수가 많으면 처리에 수 초가 걸릴 수 있습니다.
  • 파일은 브라우저 안에서만 처리됩니다. 서버로 전송되지 않습니다.
